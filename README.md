# Stochastic Terrain Navigation 

## Installation Guide

### Pre-Requisites
- [uv](https://docs.astral.sh/uv/getting-started/installation/#__tabbed_1_1)

#### Clone the Repo
```bash
git clone https://github.com/SuriyaaMM/stnav
```
#### Synchronize UV
```bash
uv sync
```
#### Activate the Virtual Environment
```bash
source stnav/bin/activate
```
